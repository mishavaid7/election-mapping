First, we have an interactive web map of the 2020 presidential election results at the state and county levels. It attempts to recreate the style used by the New York Times in their election results map.

These maps use Mapbox GL JS as the main JavaScript library. Data Sources: U.S. Census Bureau - Cartographic Boundary Files and MIT Election Data + Science Lab

This is a choropleth map. It allows us to see state-wide trends as well as individual county votes.

Zoom in to see county-level data and Zoom out to see state-level data. You can also hover and click individual states and counties to see the number of votes. Higher opacity represents a higher vote percentage for the winning candidate.

But choropleth maps are misleading because they seem to suggest that the vast majority of America votes Republican. But in reality, less populated areas at the center of the country tend to vote Republican and densely-populated coastal areas vote Democratic.

A graduated cylinder map does a better job at making this point. So to fix this, our second map is a graduated point map. It uses the same basemap as the example above but displays the results differently.

Each county is a circle on this map. The size of the circle refers to the number of votes the winning candidate got. Again, you can hover and click each circle to see the number of votes the winning candidate got.
